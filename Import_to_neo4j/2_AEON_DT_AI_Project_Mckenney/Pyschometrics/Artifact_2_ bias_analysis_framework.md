## ARTIFACT_2: BIAS_ANALYSIS_FRAMEWORK

```
BIAS ANALYSIS FRAMEWORK:

1. PUBLICATION AND AUTHOR CONTEXT

   A. PUBLICATION ANALYSIS
   - Identify political leaning and editorial positioning
   - Analyze ownership structure and potential influences
   - Examine target audience and readership demographics
   - Note publication history on similar topics
   
   B. AUTHOR ANALYSIS
   - Identify author's perspective and background
   - Analyze expertise level on the specific topic
   - Examine writing style and typical approaches
   - Note recurring patterns in author's work if available

2. FRAMING DEVICE ANALYSIS

   A. LANGUAGE CHOICES
   - Identify loaded terms and emotionally charged language
   - Analyze metaphors and analogies used
   - Examine active vs. passive voice patterns
   - Note descriptive adjectives and their connotations
   
   B. NARRATIVE STRUCTURE
   - Identify what information appears first/last
   - Analyze what is foregrounded vs. backgrounded
   - Examine cause-effect relationships as presented
   - Note chronological organization or rearrangement
   
   C. VISUAL ELEMENTS
   - Identify image selection and placement
   - Analyze captions and their relationship to images
   - Examine graphics, charts, and their presentation
   - Note what visuals are included or excluded

3. INCLUSION/EXCLUSION ANALYSIS

   A. CENTERED PERSPECTIVES
   - Identify which viewpoints receive primary attention
   - Analyze which sources are quoted directly
   - Examine which experts are considered authoritative
   - Note which aspects of the topic receive most coverage
   
   B. MARGINALIZED PERSPECTIVES
   - Identify which viewpoints receive minimal attention
   - Analyze which sources are paraphrased rather than quoted
   - Examine which potential experts are not consulted
   - Note which aspects of the topic receive little coverage
   
   C. NARRATIVE IMPACT
   - Identify how inclusion/exclusion shapes overall story
   - Analyze what alternative narratives are possible
   - Examine how different inclusion would change understanding
   - Note the effect of specific omissions

4. EMOTIONAL APPEAL ANALYSIS

   A. EMOTIONS EVOKED
   - Identify specific emotional responses triggered
   - Analyze intensity of emotional appeals
   - Examine patterns of emotional progression
   - Note contrasting emotions within the piece
   
   B. TECHNIQUES USED
   - Identify storytelling elements that generate emotion
   - Analyze language patterns that evoke feelings
   - Examine use of personal stories or examples
   - Note pacing and intensity modulation
   
   C. TARGET AUDIENCE
   - Identify who is meant to respond emotionally
   - Analyze how appeals align with audience values
   - Examine assumed emotional responses
   - Note differential appeals to different groups

5. IDEOLOGICAL MARKER ANALYSIS

   A. EXPLICIT POSITIONS
   - Identify openly stated ideological stances
   - Analyze direct policy advocacy
   - Examine clear political alignments
   - Note explicit value statements
   
   B. IMPLICIT ASSUMPTIONS
   - Identify unstated premises that reveal positioning
   - Analyze presuppositions in arguments
   - Examine normalized perspectives presented as neutral
   - Note what is treated as self-evident
   
   C. INTERNAL CONTRADICTIONS
   - Identify tensions within the presented arguments
   - Analyze inconsistencies in framing or values
   - Examine logical gaps that reveal ideological seams
   - Note where different value systems conflict
```
