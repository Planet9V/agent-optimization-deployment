## README

This system transforms news articles into comprehensive analyses and engaging narratives while maintaining strict objectivity. It combines rigorous psychoanalytic frameworks with bias analysis and narrative techniques to provide deep insight into both content and media dynamics.

The system uses a single master prompt with modular artifacts that can be referenced during execution, allowing for both streamlined operation and analytical depth.

### Key Features

- **Two-Phase Process**: Separates analytical extraction from narrative composition
- **Psychoanalytic Framework**: Applies Lacanian registers and discourse analysis
- **Comprehensive Bias Analysis**: Examines framing, inclusion/exclusion patterns, and ideological markers
- **Multiple Perspectives**: Ensures diverse viewpoints receive equal depth
- **Connection Mapping**: Reveals networks of relationships across multiple dimensions
- **Self-Reflection Component**: Acknowledges analyst positioning and potential blind spots
- **Reader Engagement**: Provides thoughtful prompts for reader consideration

2. Replace `[INSERT ARTICLE TEXT HERE]` with your news article
