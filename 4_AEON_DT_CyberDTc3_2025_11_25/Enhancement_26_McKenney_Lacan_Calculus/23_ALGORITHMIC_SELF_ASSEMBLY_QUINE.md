## 1. The Von Neumann Architecture of Self-Replication

John von Neumann proved that a machine can reproduce itself if it has:
1.  **Tape ($T$)**: The instructions (Source Code / Infrastructure-as-Code).
2.  **Constructor ($A$)**: The machine that reads $T$ and builds the output.
3.  **Copier ($B$)**: The machine that copies $T$.
4.  **Controller ($C$)**: The logic that orchestrates $A$ and $B$.

### 1.1 The Cyber-DNA
In AEON, the "Tape" is the **Graph Schema** + **Policy Code**.
*   **Mutation**: A change in the Tape (e.g., a new firewall rule).
*   **Expression**: The Constructor (Terraform/Ansible) applies the change to the world.

## 2. The Quine of Defense

A standard defense is static. A Quine Defense is dynamic.
$$ P(\text{Attack}) \to P' $$
The program $P$ reads the attack, analyzes its own source code $S(P)$, identifies the vulnerability, modifies $S(P)$ to $S(P')$, and recompiles itself into $P'$.

### 2.1 The Ouroboros Loop
1.  **Introspection**: The system reads its own binary memory.
2.  **Simulation**: It tests the patch in a sandbox (The Womb).
3.  **Metamorphosis**: It hot-swaps the running kernel with the patched version.
