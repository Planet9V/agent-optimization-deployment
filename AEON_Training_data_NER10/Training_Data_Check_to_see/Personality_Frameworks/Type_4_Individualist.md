# Enneagram Type 4: The Individualist/Romantic

## Core Personality Traits

**PERSONALITY_TRAIT**: Deep sense of being fundamentally different from others, experiences security role as uniquely burdensome
**INSIDER_INDICATOR**: Feels misunderstood by organization and may justify security violations as necessary for unique circumstances
**COGNITIVE_BIAS**: Special pleading - exempts self from security standards applied to others

**PERSONALITY_TRAIT**: Intense emotional depth and mood variability affecting security work consistency
**INSIDER_INDICATOR**: Security performance fluctuates dramatically based on emotional state
**COGNITIVE_BIAS**: Affect heuristic - security decisions heavily influenced by current emotional state

**PERSONALITY_TRAIT**: Withdrawn and introspective, prefers solitary security work over collaborative efforts
**INSIDER_INDICATOR**: Positive trait for focused security analysis; Risk for lack of oversight and isolated decision-making
**COGNITIVE_BIAS**: Introspection illusion - overconfident in self-assessment of security judgments

**PERSONALITY_TRAIT**: Envious of colleagues who seem to navigate security responsibilities without emotional struggle
**INSIDER_INDICATOR**: May sabotage peers perceived as having easier circumstances or better recognition
**THREAT_ACTOR**: Envious insider who undermines successful colleagues

**PERSONALITY_TRAIT**: Romanticizes security challenges and creates dramatic narratives around security incidents
**INSIDER_INDICATOR**: May exaggerate security threats for emotional significance or attention
**COGNITIVE_BIAS**: Availability heuristic - makes security threats more emotionally vivid than actual risk warrants

**PERSONALITY_TRAIT**: Authentic self-expression valued over organizational conformity in security approaches
**INSIDER_INDICATOR**: May implement non-standard security solutions reflecting personal aesthetic
**COGNITIVE_BIAS**: Not invented here syndrome - rejects standard security solutions for unique alternatives

**PERSONALITY_TRAIT**: Shame-prone with intense self-criticism about security deficiencies
**INSIDER_INDICATOR**: May hide security incidents or mistakes due to shame rather than career protection
**COGNITIVE_BIAS**: Negativity bias - focuses disproportionately on security failures and personal inadequacies

**PERSONALITY_TRAIT**: Creates idealized vision of perfect security implementation that reality can't match
**INSIDER_INDICATOR**: Chronic dissatisfaction with actual security posture leads to withdrawal or sabotage
**COGNITIVE_BIAS**: Nirvana fallacy - rejects practical security improvements as inadequate

**PERSONALITY_TRAIT**: Highly creative in security problem-solving, generating novel approaches to challenges
**INSIDER_INDICATOR**: Positive trait - innovative security solutions others wouldn't consider
**COGNITIVE_BIAS**: Novelty bias - may prefer creative solutions even when standard approaches more effective

**PERSONALITY_TRAIT**: Melancholic dwelling on past security failures and what might have been
**INSIDER_INDICATOR**: Difficulty moving forward from security incidents, extended rumination
**COGNITIVE_BIAS**: Hindsight bias - harsh judgment of past security decisions with current knowledge

## Integration and Disintegration

### Movement to Type 1 (Integration/Growth)

**PERSONALITY_TRAIT**: Develops disciplined and principled approach to security responsibilities
**INSIDER_INDICATOR**: Positive development - consistent security performance despite emotional fluctuations
**COGNITIVE_BIAS**: Process focus - values security methodology over emotional reactions

**PERSONALITY_TRAIT**: Becomes more objective and less emotionally reactive in security decisions
**INSIDER_INDICATOR**: Reduced sabotage risk - security judgments based on standards rather than feelings
**COGNITIVE_BIAS**: Debiasing - recognizes and corrects for emotional influences on security thinking

**PERSONALITY_TRAIT**: Takes action on security issues rather than dwelling on feelings about them
**INSIDER_INDICATOR**: Positive trait - proactive security improvements rather than melancholic contemplation
**COGNITIVE_BIAS**: Action bias (positive) - implements security solutions rather than endless analysis

### Movement to Type 2 (Disintegration/Stress)

**PERSONALITY_TRAIT**: Becomes clingy and demanding of attention from security stakeholders during stress
**INSIDER_INDICATOR**: May create security crises to receive attention and validation
**THREAT_ACTOR**: Attention-seeking insider who manufactures security incidents for recognition

**PERSONALITY_TRAIT**: Manipulative behavior seeking approval while feeling unworthy of it
**INSIDER_INDICATOR**: May grant inappropriate access seeking relationship validation
**COGNITIVE_BIAS**: Reciprocity bias - expects emotional support in return for security favors

**PERSONALITY_TRAIT**: Resentful when security assistance isn't appreciated or reciprocated
**INSIDER_INDICATOR**: High sabotage risk - combination of emotional volatility and access during stress
**THREAT_ACTOR**: Resentful insider who sabotages after feeling unappreciated
