PIPELINE OVERVIEW
================================================================================

5-Step Ingestion Process:

Step 1: Document Upload & Preprocessing
  Input: Raw documents (8 formats)
  Output: Preprocessed text + metadata
  Technologies: React, Python, FileHandling

Step 2: NER11 Entity Extraction
  Input: Cleaned text
  Output: 30+ entity types with confidence scores
  Technologies: BERT, Hugging Face Transformers, Python

Step 3: OpenSPG Semantic Reasoning
  Input: Extracted entities
  Output: Knowledge graph with relationships
  Technologies: Python, NetworkX, Rule Engine

Step 4: Neo4j Graph Storage
  Input: Knowledge graph
  Output: Persistent, queryable graph database
  Technologies: Neo4j, Python, Cypher

Step 5: Intelligence Generation
  Input: Stored knowledge graph
  Output: Reports, insights, APIs, dashboards
  Technologies: Python, React, Flask, NetworkX
