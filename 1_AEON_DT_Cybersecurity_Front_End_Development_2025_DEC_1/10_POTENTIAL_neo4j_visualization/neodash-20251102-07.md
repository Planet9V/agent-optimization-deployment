## Summary
NeoDash was a comprehensive dashboard builder for Neo4j that enabled users to build graph dashboards rapidly, marketed as "build a graph dashboard in minutes." While the project is **no longer maintained** as of 2025, it remains historically significant for Neo4j visualization patterns and provides valuable architectural insights for current implementations.

## Key Information
- **Status**: Deprecated (no longer maintained since August 2025)
- **License**: Apache-2.0
- **Community**: 492 stars, 176 forks, 52 releases
- **Latest Version**: 2.4.11 (August 4, 2025)
- **Primary Languages**: TypeScript (91.9%), JavaScript (3.7%), Cypher (2.4%)

```typescript
const dashboardConfig = {
  id: 'unique-dashboard-id',
  title: 'Financial Network Analysis',
  parameters: {
    dateRange: '2025-01-01:2025-11-02',
    maxNodes: 1000
  },
  reports: [
    {
      id: 'graph-visualization',
      type: 'graph',
      query: 'MATCH (n)-[r]->(m) RETURN *',
      config: {
        nodeLabels: ['Person', 'Account', 'Transaction'],
        relationshipTypes: ['TRANSFERRED', 'OWNS'],
        layout: 'force-directed'
      }
    },
    {
      id: 'transaction-table',
      type: 'table',
      query: 'MATCH (t:Transaction) RETURN t LIMIT 100',
      config: {
        columns: ['timestamp', 'amount', 'from_account', 'to_account'],
        pagination: true
      }
    }
  ]
};
```

## Setup and Installation

### Self-Hosted Deployment
```bash
# Clone the repository
git clone https://github.com/neo4j-labs/neodash.git
cd neodash

# Install dependencies
npm install

# Build for production
npm run build

# Start development server
npm start
```

### Docker Deployment
```yaml
# docker-compose.yaml
version: '3.8'
services:
  neodash:
    build: .
    ports:
      - "3000:3000"
    environment:
      - NEO4J_URI=bolt://neo4j:7687
      - NEO4J_USER=neo4j
      - NEO4J_PASSWORD=password
    depends_on:
      - neo4j

  neo4j:
    image: neo4j:5.22
    ports:
      - "7474:7474"
      - "7687:7687"
    environment:
      - NEO4J_AUTH=neo4j/password
```

### Database Connection
```javascript
// Neo4j connection configuration
const neo4jConfig = {
  uri: process.env.NEO4J_URI || 'bolt://localhost:7687',
  user: process.env.NEO4J_USER || 'neo4j',

  maxConnectionPoolSize: 100,
```

## Migration and Alternatives

### Current Status (2025)
- **Neo4j Console Dashboards**: Official replacement with integrated visualizations
- **Neo4j Bloom**: Native graph visualization with advanced capabilities
- **Third-party Solutions**: 
  - Python Dash (Plotly)
  - React-based Neo4j visualization libraries
  - Custom web applications using Neo4j drivers

### Recommended Migration Path
1. **Neo4j Console**: Free dashboard creation within Neo4j ecosystem
2. **Neo4j Bloom**: Advanced visualization for business users
3. **Custom Development**: React/Vue.js applications with Neo4j drivers
4. **Commercial Solutions**: Enterprise visualization platforms
