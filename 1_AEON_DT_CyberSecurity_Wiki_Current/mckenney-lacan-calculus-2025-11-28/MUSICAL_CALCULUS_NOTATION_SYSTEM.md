## 2. The Note ($N$)

A single event (Log, Message) is a Note.
$$ N = (P, D, V, T) $$

### 2.1 Pitch ($P$): Sentiment/Intensity
We map **Sentiment** (Valence) to Pitch.
*   **High C (C5)**: Extreme Positive / Urgent / "Success".
*   **Middle C (C4)**: Neutral / "Info".
*   **Low C (C3)**: Extreme Negative / "Error" / "Failure".

### 2.2 Duration ($D$): Time/Length
We map **Duration** to Note Value.
*   **Whole Note**: A long-running process / A long email.
*   **Quarter Note**: A standard transaction / A chat message.
*   **Sixteenth Note**: A micro-service call / A "Yes/No" reply.

### 2.3 Volume ($V$): Priority/Impact
We map **Priority** to Dynamics.
*   **Fortissimo ($ff$)**: Critical Alert / ALL CAPS.
*   **Pianissimo ($pp$)**: Debug Log / Whisper.

### 2.4 Timbre ($T$): Personality (DISC)
We map **DISC Profile** to Instrument Class.
*   **D (Dominance)**: **Brass** (Trumpet, Trombone). Loud, cutting, direct.
*   **I (Influence)**: **Woodwinds** (Flute, Oboe). Expressive, colorful, variable.
*   **S (Steadiness)**: **Strings** (Cello, Viola). Warm, continuous, supporting.
*   **C (Compliance)**: **Percussion** (Snare, Xylophone). Precise, rhythmic, sharp.
