# Kaggle Datasets

## Overview
This directory stores manually downloaded datasets from Kaggle.

## Target Datasets
1. **Big Five Personality Test**: Raw data from open source psychometric tests.
2. **Mental Health in Tech Survey**: Demographic and psychometric correlations.

## Instructions
1. Download `csv` files from Kaggle.
2. Place them in this directory or a `data/` subdirectory.
3. Write a parser script to normalize them into the shared JSON schema.
