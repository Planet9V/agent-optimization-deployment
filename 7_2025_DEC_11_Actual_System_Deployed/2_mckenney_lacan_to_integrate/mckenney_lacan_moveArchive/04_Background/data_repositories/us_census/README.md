# US Census Bureau Data Integration

## Overview
This directory is reserved for tools and documentation related to the U.S. Census Bureau API.

## Setup
1. **Get an API Key**: Register at [Census.gov API Key Request](https://api.census.gov/data/key_signup.html).
2. **Store Key**: Do not commit keys to the repo. Use an environment variable `CENSUS_API_KEY`.

## Planned Endpoints
- **American Community Survey (ACS)**: For detailed demographic data.
- **Decennial Census**: For base population counts.
